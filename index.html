Exploit Me Mobile Android

Welcome to ExploitMe Mobile by Security Compass.

This is an open source project demonstrating iPhone mobile hacking. We've developed this set of labs and a exploitable framework for you to hack mobile a application on Android

If you're a developer, you can also gain some great insight on how to securely develop mobile applications and prevent common programming pitfalls in Android applications.

Our labs will help you learn about:

  • Parameter manipulation
  • Encryption
  • Password lock screens
  • File system access
  • Insecure storage

To get started, visit the quick setup guide!
